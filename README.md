### 🚀 Products I've built and shipped

- **[Formatix AI](https://formatix.ai)** — AI platform for executive search. Turns CVs, Hogan/psychometric data and assessor notes into client-ready reports, executive profiles and candidate intelligence. **Multi-model** (Claude · Gemini · GPT) generation of **PPTX / DOCX / XLSX**, OCR, face-aware photo handling, and an **MCP server** so any AI assistant can drive it. *Live, paying customers.*
- **RecruiterClaw** — agentic recruiting automation: **source → enrich → outreach → schedule**, built on a per-user local browser-automation architecture with a chainable "skills" model.
- **Praxara** — **GxP-validated** AI agent platform for pharmaceutical **pharmacovigilance & regulatory** workflows (PV / RMP / PSUR / PSMF). Node/TS · Prisma · Postgres · React.
- **Quill** — an AI "exoskeleton" for **UK accountancy** practices: the ops layer for a modern firm.
